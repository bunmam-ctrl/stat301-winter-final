The `target_check/` directory contains exploratory data analysis (EDA) outputs related to the target variable. These files help evaluate the **distribution, missing values, and category proportions** of the target variable before modeling.  

#### Files in Folder 
- `rating_vis.png` – Visualization of the distribution of the original target variable, **`quality_star_rating`**.  
- `na_target.rda` – Data object summarizing missing values in the target variable.  
- `category_pie.png` – Pie chart showing the proportion of the transformed binary target variable, **`quality_star_category`**.
