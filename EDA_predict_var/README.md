Each file in `EDA_predict_var/` contains **distribution plots of predictor variables on training set**, helping to visualize their spread, trends, and potential relationships before model fitting. 

### Files in Folder 

- `care_trans_dis.png` – Distribution plot for care coordination and information transfer metrics.  
- `delivery_dis.png` – Distribution plot for timeliness and care delivery variables.
- `finance_dis.png`– Distribution plot for financial-related predictors.  
- `improve_dis.png`– Distribution plot showing patient improvement-related measures.  
- `med_safe_dis.png` – Distribution plot for medicatio and safety compliance variables.  
- `medicare_index_dis.png` – Distribution plot for Medicare index values.  
- `perform_dis.png` – Distribution plot for performance-related predictors.
