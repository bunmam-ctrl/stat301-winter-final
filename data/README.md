This folder contains **raw, transformed, and tidy data files** of the [Home Health Care Agencies Dataset](https://data.cms.gov/provider-data/dataset/6jpm-sxkc#data-table), along with a codebook that documents the dataset structure and variables. These files are essential for **data preprocessing, cleaning, and analysis**.  

#### Files in Folder   

- `HHS_provider_raw.csv` – Raw dataset in CSV format before any preprocessing.  
- `hhs_provider.rds`– Transformed dataset stored in RDS format for efficient loading.  
- `hhs_tidy.rds`– Tidy version of the transformed dataset for analysis.  
- `HHS_Data_codebook.pdf` – Codebook documenting dataset variables, definitions, and data collection details.
