The `4_model_analysis/` directory contains scripts for **evaluating and comparing model performance**. These scripts analyze key metrics to determine the best-performing model.  

Both scripts perform similar analyses but differ based on the preprocessing approach used:  

- `basic.R` – Model performance analysis for models trained with the **basic recipe** (minimal preprocessing).  
- `complex.R` – Model performance analysis for models trained with the **complex recipe** (advanced feature transformations).
