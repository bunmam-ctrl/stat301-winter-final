Each file in `final_fitting/` contains **artifacts related to the final model training and evaluation**, including fitted model objects, performance metrics, and visualization outputs. These files support **model interpretation, validation, and refinement**.  

#### Files in Folder

- `bt_hyperparameter.rda` – Saved hyperparameters for the final model.  
- `hhs_conf_mat.png` – Confusion matrix visualization for model evaluation.  
- `hhs_final_fit.rda` – Serialized final fitted model object.  
- `hhs_final_metrics.rda` – Computed performance metrics for the final model.  
- `high_roc_auc.png` – Receiver Operating Characteristic (ROC) curve with AUC visualization.
