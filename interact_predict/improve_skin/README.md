Each file in `improve_skin/` contains **multiple interaction plots** for the predictor variable `improve_skin`, visualizing its relationships with other predictors and identifying potential **interaction effects** that may influence model performance.  

#### Files in Folder

- `skin_no_interact.png` – Model performance visualization **without interaction terms**.
- `skin_perform.png` – Model performance visualization **with interaction terms** between `improve_skin` and overall performance metrics.  
- `skin_transfer.png` – Model performance visualization **with interaction terms** between `improve_skin` and information transfer metrics.  
- `skin_finance.png` – Model performance visualization **with interaction terms**between `improve_skin` and finance metrics.
