Each file in `timely_care/` contains **interaction plots** that visualize how a selected predictor variable interacts with other features. These plots help identify **interaction effects** that may impact model performance.  

#### Files in Folder 

- `timely_non_interact.png` – Model performance visualization **without interaction terms**.
- `timely_quality.png` – Model performance visualization **with interaction terms** between `timely_care` and care quality.
- `timely_performance.png` – Model performance visualization **with interaction terms** between `timely_care` and overall performance metrics.  
- `timely_improve.png` – Model performance visualization **with interaction terms** between `timely_care` and patient improvement outcomes.
