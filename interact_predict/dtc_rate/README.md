Each file in `dtc_rate/` contains **two types of interaction plots** for the predictor variable `dtc_rate`, visualizing its relationships with other predictors and identifying potential **interaction effects** that may influence model performance.  

#### Files in Folder 

- `dtc_non_interact.png` – Model performance visualization **without interaction terms**.
- `dtc_interact.png` – Model performance visualization **with interaction terms**.
