Each file in `improve_medicine/` contains **multiple interaction plots** for the predictor variable `improve_medicine`, visualizing its relationships with other predictors and identifying potential **interaction effects** that may influence model performance.  

#### Files in Folder 

- `medicine_non_interact.png` – Model performance visualization **without interaction terms**.
- `medicine_performance.png` – Model performance visualization **with interaction terms** between `improve_medicine` and overall performance metrics.  
- `medicine_transfer.png` – Model performance visualization **with interaction terms** between `improve_medicine` and information transfer metrics.
