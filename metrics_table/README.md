The `metrics_table/` directory contains **evaluation results of model performance**, summarizing key metrics for different modeling approaches.  

#### Files in Folder

- `roc_basic_reformat.png` – Summary of ROC-AUC and accuracy for the best-performing models of each model type, selected based on **ROC-AUC**, for models trained with the **basic recipe**.  
- `evaluate_complex_reformat.png` – Summary of ROC-AUC and accuracy for the best-performing models of each model type, selected based on **ROC-AUC**, for models trained with the **complex recipe**.  
- `accuracy_table_reformat.png` – Summary of F1-score, accuracy, and ROC-AUC for the best-performing logistic regression and null models, selected based on **accuracy**.
