The `predictor_check/` directory contains **exploratory data analysis (EDA) outputs** focused on assessing the quality and completeness of predictor variables. These files help identify missing data patterns and overall dataset characteristics.  

#### Files in Folder  

- `hhs_summary.rda` – Summary of the entire dataset, including the **number of observations, missingness percentage, and general data structure**.  
- `na_timely_delivery.png` – Missingness analysis for **timely delivery** of care.  
- `na_medication_safety.png` – Missingness analysis for **medication safety** measures.  
- `na_improve_outcome.png` – Missingness analysis for **patient outcome improvement**.  
- `na_finance_perform.png` – Missingness analysis for **financial performance** metrics.  
- `na_coordination_information.png` – Missingness analysis for **coordination of care and information sharing**.  
- `na_agency_service.png` – Missingness analysis for **agency-provided services**.
